To complete this task, you need to open the door to the Exit before taking the flower there.

To open the door, you simply "unlock" it by placing the correct key in the appropriate key hole. This is done by picking up the key and taking it to the there.

->

Once unlocked and open, you then can take the flower to the exit.

Some more controls, if you want to drop (or place) an object press "Q". If you want to throw it (away), press "C". You do not have to either of those in this level thoough, I am just saying....

Therefore....:

EPick up an object
QPlace or drop an object
CThrow an object
