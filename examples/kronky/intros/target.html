It is possible to throw an object you have picked up at another. This is done by:

  1. Going to where you want to throw the object you are carrying.
  2. Turn ("<" and ">" keys) so you are pointing in the direction you want the object to go.
  3. Possibly strafing may be useful too ("A" and "D" keys).
  4. Press "C" to throw what you are carrying."

A tip for this puzzle, and manoeuvering to near an edge of a platform, is to look down - using the "M" key. You can look up again - by using the "K" key.

Relevant (new) keys:

MLook Down
KLook Up
And of course...
CThrow (or chuck) the thing.
